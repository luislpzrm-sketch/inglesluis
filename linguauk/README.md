# LinguaUK v4

## IMPORTANT: Add your API key

Open `js/ai.js` and replace `YOUR_KEY_HERE` on line 5 with your Anthropic API key:

```
const KEY = 'sk-ant-api03-...your actual key here...';
```

Get your key at: https://console.anthropic.com → API Keys → Create Key

## Deploy to Vercel / Netlify

Drag and drop this folder. No build step needed — pure static files.

## Install on iPhone

1. Open your URL in Safari
2. Share → Add to Home Screen
